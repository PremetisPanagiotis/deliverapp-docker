## Requirements
- Docker
- Docker Compose

## Run locally
```bash
git clone https://github.com/PremetisPanagiotis/deliverapp-docker.git
```
```bash
cd deliverapp-docker
```
```bash
docker compose up
```

## Urls
- React: http://localhost:3000
- Django: http://localhost:8000
- MySQL: http://localhost:33060
